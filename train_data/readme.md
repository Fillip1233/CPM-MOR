# train_data

---- GAR for mf-mor data ----
- mf_high_f.npy : 欧拉仿真的真实解 仿真时间步长: 200  数据量：200 端口数量：100 数据维度 (200,100,201)
- mf_in.npy : PULSE(V1 V2 TD TR TF PW PER) 数据维度（200,7） 200是数据量
- mf_low_f.npy : SVD-MOR 选取奇异值>2的低精度解 仿真时间步长: 200  数据量: 200 端口数量: 100 数据维度 (200,100,201)
- mf_inall.npy : PULSE输入 包含100个端口200个时间步的输入信息 数据量: 200 数据维度 (200,100,201)
- 1t/ ibmpg1t 电路
- 1t/sim_100/ 100仿真步长, 各个端口输入一致的训练数据
- 1t/sim_100_diff/ 100仿真步长, 各个端口输入不一致的训练数据
- 1t/sim_100_port300_poor/ 100仿真步长,各个端口输入一致的训练数据 端口数:300 采用方法:SVD-MOR B从300列变成179列
- 1t/sim_100_port300 /100仿真步长,各个端口输入一致的训练数据 端口数: 300 采用方法: 对B进行SVD B从300列变成38列
- 1t/sim_100_port600 /100仿真步长,各个端口输入一致的训练数据 端口数: 600 采用方法: 对B进行SVD B从600列变成76列
- 1t/sim_100_port1000 /100仿真步长,各个端口输入一致的训练数据 端口数: 1000 采用方法: 对B进行SVD B从1000列变成94列
- 1t/sim_100_port1500 /100仿真步长,各个端口输入一致的训练数据 端口数: 1500 采用方法: 对B进行SVD B从1500列变成187列
- multiper 多周期的波形
- 1t/sim_100_port2000 /100仿真步长,各个端口输入一致的训练数据 端口数:2000 采用方法:对B进行SVD 取奇异值>1的部分 B从2000列变成244列
- 1t/sim_100_port2000_multiper_over /100仿真步长,各个端口输入不同的训练数据 比没有over的幅度大100倍数
- 1t/sim_100_port2000_multiper_over10 /100仿真步长,各个端口输入不同的训练数据 比没有over的幅度大10倍数
- 1t/sim_100_port2000_multiper_over3 /100仿真步长,各个端口输入不同的训练数据 比没有over的幅度大3倍数
- 1t/sim_100_port2000_multiper_over3_change /100仿真步长,各个端口输入不同的训练数据 比没有over的幅度大3倍数 其他的生成参数
- 1t/sim_100_port2000_multiper_sin /100仿真步长,各个端口输入不同的sin波形训练数据

- 2t/ ibmpg2t 电路
- 2t/sim_100_port1500 /100仿真步长,各个端口输入一致的训练数据 端口数:1500 采用方法:SVD-MOR 取奇异值>2的部分 B从1500列变成386列
- 2t/sim_100_port2000 /100仿真步长,各个端口输入一致的训练数据 端口数:2000 采用方法:SVD-MOR 取奇异值>3的部分 B从2000列变成696列 (展开点1e9)
- 2t/sim_100_port2000_multiper_svdB 100仿真步长,各个端口输入一致的训练数据 端口数:2000 采用方法:对B进行svd 取奇异值>1的部分 B从2000列变成41列(选择了B的10000到12000列的数据)

- 3t/ ibmpg3t 电路
- 3t/sim_100_port500_multiper /100仿真步长,各个端口输入一致的训练数据 端口数:1000 采用方法:对B进行SVD 取奇异值>1的部分 B从500列变成15列
- 3t/sim_100_port1000_multiper /100仿真步长,各个端口输入一致的训练数据 端口数:1000 采用方法:对B进行SVD 取奇异值>1的部分 B从1000列变成55列
- 3t/sim_100_port1500_multiper /100仿真步长,各个端口输入一致的训练数据 端口数:1500 采用方法:对B进行SVD 取奇异值>1的部分 B从1500列变成126列
- 3t/sim_100_port2000_multiper /100仿真步长,各个端口输入一致的训练数据 端口数:2000 采用方法:对B进行SVD 取奇异值>1的部分 B从2000列变成138列
- 3t/sim_100_port4000_multiper /100仿真步长,各个端口输入一致的训练数据 端口数:2000 采用方法:对B进行SVD 取奇异值>1的部分 B从4000列变成225列

- 4t/ ibmpg4t 电路
- 4t/sim_100_port1500_multiper /100仿真步长,各个端口输入一致的训练数据 端口数:1500 采用方法:对B进行SVD 取奇异值>1的部分 B从1500列变成163列
- 4t/sim_100_port2000_multiper /100仿真步长,各个端口输入一致的训练数据 端口数:2000 采用方法:对B进行SVD 取奇异值>1的部分 B从2000列变成413列
- 4t/sim_100_port2000_multiper /100仿真步长,各个端口输入一致的训练数据 端口数:2000 采用方法:SVD-MOR 取奇异值>2的部分 B从2000列变成477列

- 6t/ ibmpg6t 电路
- 6t/sim_100_port1000_multiper_diff /100仿真步长,各个端口输入一致的训练数据 端口数:1000 采用方法:对B进行SVD 取奇异值>1的部分 B从1000列变成81列 (选取了170700到 171700列的数据)
- 6t/sim_100_port2000_multiper_diff /100仿真步长,各个端口输入一致的训练数据 端口数:2000 采用方法:对B进行SVD 取奇异值>1的部分 B从2000列变成165列 (选取了170700到 172700列的数据)
